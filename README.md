# NodeServe

A minimal static file server built using the Node.js HTTP module.

This project serves static assets such as HTML, CSS, JavaScript, images, and other files without using any external frameworks.

## Features

* Static file serving
* HTML, CSS and JavaScript support
* Image support
* MIME type handling
* Custom 404 page
* Built with Node.js core modules only

## Tech Stack

* Node.js
* HTTP Module
* File System (fs)
* Path Module

## Project Goal

The purpose of this project is to understand how static file servers work internally and how HTTP requests are handled in Node.js without Express.js or other frameworks.

## Supported Files

* `.html`
* `.css`
* `.js`
* `.png`
* `.jpg`
* `.jpeg`
* `.svg`
* `.ico`
* `.json`
